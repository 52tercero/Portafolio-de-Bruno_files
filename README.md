# Portfolio Personal — Nombre Apellido

Proyecto de ejemplo: portfolio personal construido con HTML5 y CSS externo.

Archivos principales:
- `index.html` — Página principal con secciones: Sobre mí, Formación, Proyectos y Contacto.
- `styles.css` — Estilos externos para diseño responsivo.

Cómo ver localmente:
1. Abrir `index.html` en el navegador (doble clic o abrir con navegador).

Despliegue en GitHub Pages (resumen):
1. Crear un repositorio público en GitHub e iniciar con estos archivos.
2. Hacer push al repositorio.
3. En la configuración del repositorio -> Pages, seleccionar la rama `main` (o `gh-pages`) y la carpeta `/`.
4. Esperar la URL proporcionada por GitHub Pages.

Si quieres, puedo:
- Inicializar el repositorio git local, crear commits y mostrar los comandos.
- Ayudarte a subirlo a GitHub y activar GitHub Pages.
